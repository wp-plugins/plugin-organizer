== Description ==

This plugin allows you to change the order that your plugins are loaded.

== Installation ==

1. Extract the downloaded Zip file.
2. Upload the 'plugin_organizer' directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. There will be a new menu item under settings in the WordPress admin called Organize Plugins.
